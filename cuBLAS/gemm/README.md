# cuBLAS Level-3 APIs - `cublas<t>gemm`

## Description

This code demonstrates a usage of cuBLAS `gemm` function to compute a matrix-matrix product

```
A = | 1.0 | 2.0 |
    | 3.0 | 4.0 |

B = | 5.0 | 6.0 |
    | 7.0 | 8.0 |
```

Sample example output:

```
A
1.00 3.00 
2.00 4.00 
=====
B
5.00 7.00 
6.00 8.00 
=====
C
23.00 31.00 
34.00 46.00 
=====
```
